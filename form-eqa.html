ข้อมูลเคส

① แก้เฉพาะหน้า + ผลกระทบผู้ป่วย

ISO §8.7(a) — react / contain

② วิเคราะห์สาเหตุ + ขอบเขต

ISO §8.7(b) — root cause + extent

③ การแก้ที่รากเหง้า

ISO §8.7(c) — corrective action

④ ทวนสอบประสิทธิผล + ปิดงาน

ISO §8.7(d) — ปิดได้เมื่อ Effectiveness = ยืนยันได้ผล

ℹ️ จะปิด (Closed) ได้ต่อเมื่อ Effectiveness = "ยืนยันได้ผล" — ถ้ายังรอทวนสอบรอบถัดไป ให้คง Pending
