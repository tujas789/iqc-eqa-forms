IQC/EQA Corrective Action

ระบบบันทึก+ติดตามการแก้ไขผล IQC/EQA · กลุ่มงานเทคนิคการแพทย์ รพ.พิษณุโลกฮอสพิทอล

🧪 โหมดทดลอง (DEMO) — เปิดไฟล์ในเครื่อง · ข้อมูลไม่ถูกบันทึกจริง

📊 บันทึกผล EQA/ILC →

กรอก z-score จากใบรายงาน → ระบบตัดเกรดอัตโนมัติ → ชี้ analyte ที่ต้องเปิด corrective

📝 EQA Corrective Action →

บันทึกการแก้ไขผลที่ไม่ผ่านเกณฑ์ (CAPA ISO 15189 §8.7) — สาเหตุ → ปฏิบัติการแก้ไข → ปิดเคส

🧫 IQC Corrective Action →

บันทึกเคส IQC (Internal QC) ที่ไม่อยู่ในเกณฑ์ + การแก้ไข (CAPA ISO 15189 §8.7) — สภาพปัญหา → ทบทวน → ปฏิบัติการแก้ไข → ปิดเคส

📋 รายการ / ปิดงาน IQC →

ดูเคส IQC ที่ยังไม่ปิด (Status ≠ Closed) → คลิกเคส → กรอก CAPA §8.7 ที่ขาด → ปิดงาน

เปิดไฟล์นี้ในเครื่อง = โหมด DEMO (ลองหน้าตา/flow ได้ · ไม่เขียนชีตจริง)
