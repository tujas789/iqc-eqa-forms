กลุ่มงานเทคนิคการแพทย์ รพ.พิษณุโลกฮอสพิทอล

IQC/EQA Corrective Action

ระบบบันทึกและติดตามการแก้ไขผลการตรวจวิเคราะห์ที่ไม่อยู่ในเกณฑ์ยอมรับ

โหมดทดลอง (DEMO) — กำลังเปิดไฟล์จากคอมพิวเตอร์ (file:// หรือ localhost) การทำรายการในโหมดนี้จะไม่ถูกบันทึกลงในฐานข้อมูลระบบงานจริง

EQA (External Quality Assessment)

การควบคุมคุณภาพโดยองค์กรภายนอกและการเปรียบเทียบระหว่างห้องปฏิบัติการ
📊
บันทึกผล EQA/ILC
กรอก z-score จากใบรายงานประเมินผล เพื่อวิเคราะห์เกรดการประเมินและชี้เป้าทดสอบที่ต้องทำการแก้ไข
→
📝
EQA Corrective Action
บันทึกปฏิบัติการแก้ไขเมื่อตรวจพบผล EQA ที่ไม่ผ่านเกณฑ์ (CAPA ISO 15189 §8.7) จากสาเหตุไปจนถึงขั้นตอนการปิดงาน
→

IQC (Internal Quality Control)

การควบคุมคุณภาพภายในเพื่อเฝ้าระวังความถูกต้องแม่นยำของผลการทดสอบประจำวัน
🧫
IQC Corrective Action
บันทึกเหตุการณ์ IQC ที่ไม่อยู่ในเกณฑ์การควบคุม พร้อมแนวทางการประเมิน วิเคราะห์ และการดำเนินการแก้ไขเบื้องต้น
→
📋
รายการ / ปิดงาน IQC
ติดตามเคสควบคุมคุณภาพภายใน (IQC) ที่ยังไม่ได้รับการปิดงาน หรือต้องการการระบุ CAPA เพิ่มเติมเพื่อบันทึกเสร็จสิ้นงาน
→

วิเคราะห์และรายงานผล

รายงานสถานะการแก้ไข สถิติความก้าวหน้า และวิเคราะห์แนวโน้มเพื่อพัฒนางาน
📈
Dashboard สรุปภาพรวม
ภาพรวมเชิงสถิติของการดำเนินงานแก้ไข CAPA อัตราการปิดงานสำเร็จ และสัดส่วนหมวดหมู่ปัญหาทั้งหมดที่เกิดขึ้น
→

เปิดไฟล์นี้ในเครื่อง = โหมดทดลอง (DEMO) ทดสอบหน้าตาและการใช้งานในระบบแบบออฟไลน์ได้โดยไม่มีการบันทึกจริง
สำหรับการบันทึกจริงจะใช้งานผ่าน GitHub Pages โดยระบบจะยิงข้อมูลไปยัง Apps Script backend API โดยตรง
