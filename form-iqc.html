IQC Corrective Action — บันทึกเคสไม่ผ่าน

เคส IQC (Internal QC) ที่ไม่อยู่ในเกณฑ์ + การแก้ไขตาม ISO 15189:2022 §8.7 · กลุ่มงานเทคนิคการแพทย์ รพ.พิษณุโลกฮอสพิทอล

โหมดแก้ไข

ข้อมูลเคส

① สภาพปัญหา

กฎ Westgard ที่ละเมิด + ประเภท error

② ผลการทบทวน IQC

ตรวจน้ำยา / control / calibration ว่าถูกต้องหรือไม่

③ ปฏิบัติการแก้ไข (Correction)

ISO §8.7(a) — react / contain · เลือกได้หลายข้อต่อหมวด (ปิดงานต้องมีอย่างน้อย 1)

④ สาเหตุ + ขอบเขต + ผลกระทบผู้ป่วย

ISO §8.7(a,b) — root cause + extent + risk

⑤ สรุป / ทวนสอบ / ปิดงาน

ISO §8.7(c,d) — corrective action + ทวนสอบประสิทธิผล · ปิดได้เมื่อ Effectiveness = ยืนยันได้ผล

ℹ️ จะปิด (Closed) ได้ต่อเมื่อ Effectiveness = "ยืนยันได้ผล" — ถ้ายังรอทวนสอบรอบถัดไป ให้คง Pending
